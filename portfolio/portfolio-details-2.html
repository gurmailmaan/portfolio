Project information

  • Category: Php and MySQL
  • Client: Course Assignment
  • Project date: November 2022
  • Project URL: Games Collection

This project was a catalogue of games I have played over the years, made with Php and MySQL. All the information for each game will be entered into the database. I used different database fields. I used fields such as the title of the game, the genre of the game, the platform of the game, modes of the game, the Writer of the game, the Engine which runs the game, the year of release, the poster of the game, etc. Then I was be able to categorize the game into different categories like platform, year, modes, engine, etc. I was also add a brief description of each game.
